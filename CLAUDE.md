# Personal Site Project Memory

## GitHub Repository
- **URL**: https://github.com/asaleem9/personal-site
- **Configured Remote**: origin (fetch & push)

## Project Overview
This is Dave's personal portfolio site built with Next.js, featuring a brutalist design aesthetic.

## Key Features
- Hero section with 3D background
- Smooth scroll animations with GSAP
- Responsive mobile layouts
- Dark/light theme support
- Blog and GitHub integration

## Tech Stack
- Next.js
- React with TypeScript
- Tailwind CSS
- GSAP for animations
- Three.js for 3D scenes
